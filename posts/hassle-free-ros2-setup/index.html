Installing Ros2

This script is designed to save time by automating the process of copying content from the official page and identifying potential errors. It’s an interactive script that automatically detects your operating system and presents a list of compatible ROS2 distributions for you to choose from. The script will also guide you through sourcing the required paths and offer to create a basic workspace at your desired location to help you get started.

Please note that ROS2 versions are specific to Ubuntu versions. For example, ROS2 Humble is only supported on Ubuntu Jammy and its derivatives. If the script is unable to detect a supported operating system, you may manually override the auto-selection of the ROS2 distribution. However, proceed with caution and be aware of the risks involved.

  1. Log into Linux Mint and Open a terminal by pressing ctrl + alt + t and Install the git by running:
sudo apt install git
  1. Now clone my ros2-installer repo and go inside the folder by running:
git clone https://github.com/mohammedrashithkp/ros2-installer.git
&& cd ros2-installer
  1. Run the Installation Script after giving necessary permission using:
sudo chmod +x ros2-installer.sh
&& sudo ./ros2-installer.sh
